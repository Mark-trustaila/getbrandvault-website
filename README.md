# BrandVault — getbrandvault.com

Marketing landing page for BrandVault, the trademark portfolio management platform built on LawPanel.

## Structure

```
/
├── index.html          Main landing page
├── privacy.html        Privacy policy
├── favicon.svg         SVG favicon
├── favicon.ico         ICO favicon (add manually)
├── og-image.png        Open Graph image (add manually)
└── images/
    └── brandvault1.png Dashboard screenshot
```

## Deployment

Connected to Vercel. Deploys automatically on push to main.

Custom domain: getbrandvault.com
